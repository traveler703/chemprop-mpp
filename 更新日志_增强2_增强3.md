# 更新日志：增强 2 与增强 3

本次更新在已完成 **Lipophilicity 双数据集扩展** 的基础上，继续完成两个面向最终 research report 的增强：

1. **Random Forest baseline**：加入 `RDKit descriptors + RandomForestRegressor`，用于与 Chemprop MPNN 做模型范式对比。
2. **Random split vs Scaffold split**：加入 Bemis-Murcko scaffold split，用于分析模型面对新化学骨架时的泛化能力。

这些增强让项目从“单模型复现实验”升级为一个更完整的分子性质预测研究项目，可支撑最终 10 页以内 research report 或课堂展示。

## 2. 项目完整运行流程

### 2.1 一键运行主流程

推荐从项目根目录运行：

```bash
python scripts/run_all.py
```

该命令会依次执行：

1. 准备 ESOL 与 Lipophilicity 数据集
2. 生成 random split 与 scaffold split
3. 训练 Chemprop baseline
4. 运行 Random Forest baseline
5. 运行数据规模实验
6. 生成分子案例预测
7. 生成结果图表
8. 生成 PPT

如果希望跳过耗时较长的数据规模实验：

```bash
python scripts/run_all.py --skip-scales
```

如果只想准备数据：

```bash
python scripts/run_all.py --only-data
```

如果只想训练 Chemprop baseline：

```bash
python scripts/run_all.py --only-train
```

如果暂时跳过 Random Forest baseline：

```bash
python scripts/run_all.py --skip-rf
```

## 7. 最终 Research Report 框架

作业要求允许提交 **10 页以内 research report 或 PPT**。如果选择 report，建议采用下面结构。

### 7.1 Introduction

建议内容：

- AI for Science 背景
- 分子性质预测在药物发现中的意义
- 为什么水溶解度和脂溶性重要
- 本项目研究问题

核心研究问题可写为：

> How does molecular graph learning perform under different scientific data conditions, including data scale, molecular property type, model paradigm, and chemical generalization split?

### 7.2 Dataset

建议内容：

- ESOL 数据集：水溶解度 LogS
- Lipophilicity 数据集：脂溶性 logD
- SMILES 表示
- random split 与 scaffold split 的区别

推荐表格：

| Dataset | Task | Molecules | Target | Split |
|---|---|---:|---|---|
| ESOL | Solubility prediction | 1128 | LogS | random / scaffold |
| Lipophilicity | Lipophilicity prediction | 4200 | logD | random / scaffold |

### 7.3 Methods

建议内容：

- Chemprop MPNN：从分子图结构中学习表示
- Random Forest baseline：基于 RDKit 手工描述符
- RDKit descriptors：MolWt、MolLogP、TPSA、HBD、HBA、RotB、RingCount、FractionCSP3
- 评价指标：RMSE、MAE、R2

### 7.4 Experiments and Results

建议分成四组实验：

1. Chemprop baseline on ESOL and Lipophilicity
2. Data scale experiment
3. Chemprop vs Random Forest
4. Random split vs Scaffold split

推荐图表：

```text
results/predicted_vs_true.png
results/lipo_predicted_vs_true.png
results/data_scale_rmse.png
results/lipo_data_scale_rmse.png
results/model_comparison.png
results/molecule_examples.png
```

### 7.5 Discussion

建议讨论：

- 为什么数据规模增加会提升性能
- 为什么 ESOL 和 Lipophilicity 的 R2 不同
- Chemprop 与 Random Forest 的优劣
- scaffold split 为什么更接近真实药物发现泛化
- 数据覆盖范围和化学多样性对模型可靠性的影响

可直接使用的讨论句：

> Random split tests interpolation among similar molecules, while scaffold split tests generalization to new chemical structures. The comparison highlights that molecular property prediction should be evaluated not only by average accuracy but also by chemical generalization ability.

### 7.6 Conclusion

建议总结：

- Chemprop 在 ESOL 和 Lipophilicity 上均取得较好结果
- 更多训练数据能稳定提升模型性能
- Random Forest baseline 提供了传统机器学习对照
- scaffold split 让项目具备更强的研究分析价值
- AI for Science 中数据质量、结构和科学先验会共同影响模型边界
